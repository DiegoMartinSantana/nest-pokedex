# Execute in development

1. Clone repository
2. Execute 
```
npm install

```
3.  Nest Cli install
```
npm i -g @nesjs/cli

```
4. Up Db
```
docker-compose up -d
```

5. Refresh Bd ( GET )
```
 http://localhost:3000/api/v2/seed
 
``` 
6. Clone file __.env.template__ and rename : __.env__
